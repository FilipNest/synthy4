Synthy 4*

A collaborative multiplayer grid synthesizer by FilipNest

Squares linked to others in a chain play together. Select which beats you want to play on, pick a note, then move around the grid with arrows / keyboard to join others. Enjoy the colours. If it's lonely you can open different browsers (mute all but one or get fun phasing). WebMIDI out supported where available. Click below to start!

*in 2015 I made a similar multiplayer colour synth called Synthy. Then two more which never got to a fully releasable stage. Comments, bugs and suggestions welcome on GitHub / Twitter etc. Made with no libraries, images or samples, just standard HTML, CSS, JS (+ node.js core). Colours via Scriabin. Dreaming of a pretty world to put this in rather than a grid... Anyway! Thank you Glitch for hosting.
