Synthy 4*

A collaborative multiplayer grid synthesizer by by FilipNest

Squares linked to others in a chain play together. Select which beats you want to play on, pick a note, then move around the grid to join others. Enjoy the colours. If it's lonely you can open different browsers (recommended to mute all but one but the phasing is fun if you don't). Click below to start!

*in 2015 I made a similar multiplayer colour synth called Synthy. Then two more which never got to a fully releasable stage. Comments, bugs and suggestions welcome on GitHub / Twitter etc. Made with no libraries, images or samples, just standard HTML, CSS, JS (+ node.js core). Colours via Scriabin. Would probably be more fun with better sounds so could swap in a better synth system (or web midi which is what Synthy 1 did), but that's for another day. Or an actual pretty world rather than a grid... Anyway! Thank you Glitch for hosting.
